# Method: sessions.create

Cria uma nova sessão.

---

## HTTP Request

```http
POST https://jules.googleapis.com/v1alpha/sessions
```

A URL utiliza a sintaxe **gRPC Transcoding**.

---

## Request Body

O corpo da requisição deve conter uma instância de **Session**.

---

## Response Body

Se bem-sucedida, a resposta conterá uma nova instância criada de **Session**.
