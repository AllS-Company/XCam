# Biblioteca de Prompts do Jules

Esta biblioteca contém uma coleção de prompts de tarefa pré-fabricados que capacitam o Jules a realizar uma variedade de tarefas de engenharia de software. Cada prompt é projetado para uma finalidade específica e pode ser usado como ponto de partida para suas próprias tarefas.

Consulte os arquivos individuais no diretório `../prompts/` para ver o conteúdo completo de cada prompt.

## Índice da Biblioteca

| Arquivo do Prompt | Propósito | Quando Usar |
| :--- | :--- | :--- |
| **`task_generate_prompt_from_description.md`** | Gerar um novo prompt de alta qualidade a partir de uma descrição do usuário. | Quando você tem uma ideia para um novo prompt e quer ajuda para escrevê-lo. |
| **`task_audit_repo.md`** | Realizar uma auditoria abrangente e baseada em evidências de um repositório. | Ao iniciar em um novo projeto ou antes de uma grande refatoração. |
| **`task_analyze_and_improve_ui_ux.md`** | Analisar e sugerir melhorias na UI/UX de um frontend. | Para obter uma nova perspectiva e melhorar a experiência do usuário. |
| **`task_harden_repo_initial.md`** | Realizar um fortalecimento inicial e completo de um repositório. | Em um repositório novo ou legado para estabelecer as melhores práticas de CI/CD e testes. |
| **`task_harden_repo_iterative.md`** | Realizar melhorias contínuas e iterativas em um repositório maduro. | Para manutenção regular, correção de instabilidades e melhoria da cobertura de testes. |
| **`task_fix_and_refine.md`** | Transformar um protótipo em uma aplicação de nível de produção. | Para lidar com dívida técnica e tornar um protótipo mais robusto. |
| **`task_build_from_plan.md`** | Implementar os próximos passos lógicos a partir de um plano de projeto. | Quando um projeto tem um roteiro definido, mas precisa de implementação. |
| **`task_update_dependencies.md`** | Atualizar as dependências de um projeto para as versões mais recentes. | Para manter o projeto seguro e atualizado. |
| **`task_curate_repo.md`** | Analisar e fazer melhorias seguras em um repositório desconhecido. | Para curadoria em massa ou triagem de um grande número de repositórios. |
