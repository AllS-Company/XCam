# XCam App

This is the main frontend application for the XCam project. It provides the user interface for browsing and viewing live streams.

## Running Locally

To run this application locally, you can use a simple HTTP server.

1.  **Navigate to the `xcam-app` directory:**
    ```bash
    cd xcam-app
    ```

2.  **Start a simple HTTP server.** If you have Python 3 installed, you can use:
    ```bash
    python -m http.server
    ```
    If you have Node.js installed, you can use the `serve` package:
    ```bash
    npx serve
    ```

3.  **Open your browser** and navigate to the local address provided by the server (e.g., `http://localhost:8000`).
